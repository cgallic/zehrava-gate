# Agent Sentinel
**PagerDuty for AI Agents.**

When an agent fails, loops, times out, or goes off-rails — Sentinel detects it in real time, alerts the right human, and provides replay context for fast resolution.

## Handoff Protocol
- Specs go in `/docs/`
- Code goes in `/src/`
- Decisions logged in `/decisions/YYYY-MM-DD-decision.md`
- No credentials in this repo — secrets stay in `.env` on each server
